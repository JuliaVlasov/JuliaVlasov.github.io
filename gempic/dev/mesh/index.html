    GEMPIC.TwoDGrid — Type
    TwoDGrid( dimx, nx, dimy, ny)

    Generate a cartesians mesh on rectangle dimxx dimy with nx x ny points

    • nx : indices are in [1:nx]
    • ny : indices are in [1:ny]
    • dimx = xmax - xmin
    • dimy = ymax - ymin
    • x, y : node positions
    • dx, dy : step size
